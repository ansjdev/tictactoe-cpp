# tictactoe-cpp
An simple and funny tic tac toe made in c++!

## How to install?
- Clone this repository: `git clone https://github.com/ansjdev/tictactoe-cpp.git`
- Compile the code: `g++ .\main.cpp`
- Run: `.\main.exe`

#### Only tested in Windows.
